Matrix filler

ni nj mi v v v v v v .. .. v v v v v v .. .. v v v v v v .. .. v v v v v v .. .. v v v v v v .. mj

Parameters:

  • A must be a matrix.
    (matrix pointer)

  • v must be an any.
    (value used to fill the matrix)

  • mi must be a const index.

  • mj must be a const index.

  • ni must be a const index.

  • nj must be a const index.

var fill = function ( A, mi, mj, ni, nj, v ) { var r, c, Ar; for ( r = mi ; r < mj ; ++r ) { Ar = A[r]; for ( c = ni ; c < nj ; ++c ) { Ar[c] = v; } } }; exports.fill = fill;
